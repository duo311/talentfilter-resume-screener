
# TalentFilter - AI Resume Screener 🔍

AI-powered tool to screen and rank resumes against job descriptions using LLMs.

## Features
- 🤖 **Multiple AI Providers**: Google Gemini (FREE), Anthropic Claude, OpenAI GPT
- 📄 **Resume Support**: PDF, DOCX, TXT files
- 📊 **Smart Scoring**: Skills, Experience, Education, Cultural Fit
- 💾 **Export Results**: CSV and JSON formats

## Quick Start

```bash
# Clone repo
git clone https://github.com/duo311/talentfilter-resume-screener.git
cd talentfilter-resume-screener

# Install dependencies
pip install -r requirements.txt

# Run app
streamlit run app.py
